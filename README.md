# 💳 Fawry Transaction Analysis Dashboard

## 📌 Project Overview

The Fawry Transaction Analysis Dashboard is an interactive Excel dashboard developed to analyze digital payment transactions, revenue performance, user activity, merchant performance, and payment behavior across multiple transaction categories.

The project transforms raw transaction data into actionable business insights through data cleaning, transformation, aggregation, and visualization using Excel and Power Query.

The dashboard helps stakeholders monitor business performance, identify transaction trends, evaluate payment methods, and support data-driven decision-making.

---

## 🎯 Business Objectives

- Monitor overall transaction performance.
- Analyze revenue generation across product categories.
- Evaluate payment method performance.
- Track user and merchant activity.
- Identify transaction patterns by device type and time of day.
- Analyze transaction success rates.
- Support strategic business decisions through data insights.

## 🔍 Key Insights

### Revenue Performance
- Total revenue exceeded **24.6 Million**.
- Revenue performance varied across months.
- Product category analysis identified key revenue drivers.

### Payment Methods
- Bank Transfer generated the highest revenue.
- Debit Card and UPI showed strong transaction activity.
- Wallet Balance transactions contributed a smaller share of revenue.

### Device Analysis
- Android users generated the highest number of transactions.
- iOS ranked second in transaction volume.
- Web transactions represented the smallest share.

### Merchant Performance
- Top merchants contributed significantly to overall revenue.
- Revenue concentration highlighted high-performing business partners.

### Transaction Behavior
- Night-time transactions recorded the highest activity.
- Most transactions were successfully completed.
- Failed transactions represented only a small percentage of total transactions.
